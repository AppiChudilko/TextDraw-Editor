# TDE - TextDraw Editor
By Appi

# Features

- Making textdraws in game easy (box and text textdraws).
- Move the TextDraw with mouse.
- Adaptive resolution.
- Exporting TextDraws in lua or csharp file.

# Use

- Press F1 :)
